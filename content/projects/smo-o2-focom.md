FOCOM (Federated O-Cloud Orchestration and Management) is the SMO-side component that
talks to the O-Cloud IMS over the O2 interface. This is an MVP implementation built to
test provisioning flows before contributing to the OSC upstream.

**Stack:** Python, Flask, O2ims REST API
